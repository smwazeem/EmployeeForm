WindowsFormsApp1 folder has whole application codes 

EmployeeDB.bacpac is the EmployeeDB database. you can import this and execute in MS SQL.

Open into WindowsFormsApp1 folder and open WindowsFormsApp1 microsoft Visual Studio file in visual studio.
 then click on start button on the top. 

MainForm.cs file has the connectionString = "Data Source=(LocalDb)\\MSSQLLocalDB;Initial Catalog=EmployeeDB;Integrated Security=True";
